# Linguagem de Programação I

Profª Dra. Adriana da Silva Jacinto  

Repositório criado para postar atividades e exercícios da disciplina **Linguagem de Programação I**, segundo semestre do curso de Tecnologia em Banco de Dados da Fatec São José dos Campos – Professor Jessen Vidal.

## 🔧 Tecnologias utilizadas
- **Java** ![Java logo](iTurn0image1)  
- **Programação Orientada a Objetos (POO)** ![POO](iTurn0image10)  
- **IDE: IntelliJ IDEA** ![IntelliJ logo](iTurn0image4)  
- **Interface gráfica: SceneBuilder (JavaFX)** ![SceneBuilder logo](iTurn0image7)  

## 🎯 Objetivo
Este repositório reúne exercícios e projetos desenvolvidos ao longo da disciplina, com o intuito de:
- Praticar conceitos fundamentais de orientação a objetos.  
- Desenvolver aplicações Java utilizando boas práticas de programação.  
- Explorar a criação de interfaces gráficas com JavaFX e SceneBuilder.  

## 📂 Estrutura
Cada pasta corresponde a um exercício ou projeto específico, contendo:
- Código-fonte em Java.  
- Arquivos de configuração da IDE.  
- Interfaces gráficas quando aplicável.
